Physics 1251 Programming Lab: Wave Interference!

Este laboratorio de programación está diseñado para ilustrar la idea de interferencia de ondas. Esto sucede debido al principio de "superposición". Para las ondas sonoras, esta palabra de 20 dólares simplemente significa que si hay dos altavoces en una habitación grande o en un espacio abierto, la amplitud de las ondas sonoras en un punto particular de esa habitación es la suma de las ondas de los dos altavoces.

Paso 4. Piense en las matemáticas involucradas

Si solo hubiera el altavoz #1, la amplitud instantánea $S_1$ en (x,y) sería la siguiente: $$S_1 = A_1 \sin (k_1 r_1 - \omega_1 t)$$

donde $r_1$ es la distancia entre el hablante # 1 y el punto (x,y).

Del mismo modo, la amplitud instantánea en (x, y) del solo altavoz #2 sería la siguiente: $$S_2 = A_2 \sin (k_2 r_2 - \omega_2 t)$$

donde $r_2$ es la distancia entre el hablante #2 y el punto (x, y).

Si tiene ambos altavoces, la amplitud en un punto particular de la habitación sería la siguiente: $$S_1 + S_2 = A_1 \sin (k_1 r_1 - \omega_1 t) + A_2 \sin (k_2 r_2 - \omega_2 t)$$

Como se explica en la hoja de trabajo, la siguiente identidad de trigonometría es muy útil: $$\sin a + \sin b = 2 \cos \left(\frac{a-b}{2} \right) \sin \left( \frac{a+b}{2} \right)$$

Como se discutió en la hoja de trabajo, cuando asume $A_1= A_2$, $\omega_1 = \omega_2$, y $k_1 = k_2$ uno encuentra esto: $$S_1 + S_2 = 2 A_1 \cos \left(\frac{k (r_1-r_2)}{2} \right) \sin \left(\frac{k (r_1+ r_2) - 2\omega t }{2}\right)$$

y la interferencia constructiva / destructiva depende de si el término coseno resulta ser igual a cero o uno.

Paso 7. ¡Aproximado!

Resulta difícil trabajar con la expresión anterior: $$|r_1 - r_2| = \frac{n\lambda}{2}$$ $$\sqrt{(x+d)^2 + y^2} - \sqrt{(x-d)^2 + y^2} = \frac{n\lambda}{2}$$

Realmente nos gustaría poder resolver para y, pero no hay una forma sencilla de hacerlo. Si intenta cuadrar ambos lados, por ejemplo, esto no ayuda mucho para aislar yy x.

Cuando estás atascado así, a veces lo mejor que puedes hacer es aproximar. Observe el siguiente truco si $a \gg b$, $$\sqrt{a^2 + b^2 } = a \sqrt{1 + \frac{b^2}{a^2} } \approx a \left( 1 + \frac{1}{2} \frac{b^2}{a^2} \right)$$

Utilice este truco para demostrar que las líneas de interferencia deconstructiva siguen esta fórmula: $$ y = \pm \frac{4 d}{n \lambda} x $$
